Guias detalhados (exemplos)

1) Backup e dump de eMMC

  1. Desligue o aparelho. Abra o case com cuidado.
  2. Localize a eMMC. Use adaptador USB-eMMC ou leitor.
  3. Conecte ao PC Linux. Use lsblk para identificar o dispositivo.
  4. Execute: sudo dd if=/dev/sdX of=backup_emmc.img bs=4M status=progress
  5. Verifique checksum: sha256sum backup_emmc.img

2) Flash Amlogic via USB

  1. Coloque box no modo FEL ou USB de boot (instruções por modelo).
  2. Use ferramenta oficial ou aml-flash para gravar a imagem.
  3. Verifique logs e reenvie se houver erros.

3) Recuperar U-Boot via TTL

  1. Localize pinos TTL. Use adaptador USB-TTL 3.3V.
  2. Conecte TX/RX/GND. Não conectar 5V.
  3. Abra terminal (115200 8N1). Interrompa boot e grave novo U-Boot.
